## 替换空格
### 描述：实现一个函数，把字符串 s 中的每个空格替换成"%20"
```java
class Solution {
    public String replaceSpace(String s) {
         int len=s.length();
         int count=0;
         for(int i=0;i<len;i++){
             if(s.charAt(i)==' '){
                 count++;
             }
         }
         char res[]=new char[len+count*2];
         int  index=0;
         for(int i=0;i<len;i++){
             if(s.charAt(i)!=' '){
                 res[index]=s.charAt(i);
                 index++;
             }else{
                 res[index++]='%';
                 res[index++]='2';
                 res[index++]='0';
             }
         }
         return new String(res);   
    }
}
```
